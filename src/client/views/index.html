Evaluate a News Article with Natural Processing Language

This tool will evaluate an URL, e.g. a news article, with Natural Processing Language using the MeaningCloud API.

The URL will be investigated based on the following key figures:

  • Score Tag: it analyzes the polarity of the text in the URL, possible values can be strong positive to strong negative or without polarity.
  • Agreement: marks the agreement between the sentiments detected in the url, it has two possible values: AGREEMENT for same polarity and DISAGREEMENT
  • Subjectivity: marks the subjectivity of the text, the values can be OBJECTIVE or SUBJECTIVE
  • Confidence: represents the confidence, its value is an integer number in 0-100 range
  • Irony:: indicates the irony of the text on the page, the values can be NONIRONIC and IRONIC
Your results of the Natural Processing Language will be displayed below.
